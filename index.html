Native CSS Carousel Controls

Craig · What

Native CSS Carousel Controls

::scroll-button(), ::scroll-marker, and scroll-marker-group let CSS create carousel arrows and marker navigation for a scroll container. Scroll snap is only the supporting behaviour that makes each slide land cleanly.

What is it?

It is an emerging CSS carousel feature set that adds browser-generated controls to scrollable content.

What does it do?

It can create previous/next scroll buttons and marker navigation without writing custom JavaScript.

Why use it?

It provides a native, accessible, and performant carousel solution with minimal code.

Where would you use it?

Product galleries, card shelves, image galleries, onboarding panels, feature tours, and any simple manual carousel.

What

Craig · How

How the new CSS works

  1. Build the carousel as a normal horizontal scrolling list.
  2. Add scroll-snap-type so slides stop cleanly.
  3. Add scroll-marker-group: after; for marker navigation.
  4. Use ::scroll-button(left) for the previous button.
  5. Use ::scroll-button(right) for the next button.
  6. Use ::scroll-marker on each slide for markers.
  7. Use :target-current to style the active marker.
